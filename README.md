# Lluvia de Bendición .Com — Launch Visual Final

Versión visual basada en la referencia aprobada: negro, azul noche, dorado y blanco cálido, con introducción cinematográfica, control de sonido, Radio primero, navegación móvil, contenido inline y PWA.

## Actualizar el repositorio con GitHub Desktop
1. Abre la carpeta local del repositorio `Lluvia-de-Bendicion-Final`.
2. Copia **todo el contenido** de esta carpeta encima del repositorio local.
3. Acepta **Replace/Replace the files in the destination**.
4. En GitHub Desktop escribe `Launch Visual Final`.
5. Pulsa **Commit to main** y luego **Push origin**.
6. Vercel hará el despliegue automáticamente.

## Configuración pendiente
Edita `config.js` para añadir la URL HTTPS del stream de radio, canal de video, redes y enlaces privados de donación. No publiques correos, teléfonos o identificadores personales directamente en el HTML.
